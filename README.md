# Flower Shop

A responsive landing page for an online flower shop, built with pure HTML and CSS.

## Features

- **Hero Banner** — "Experience the Joy of Fresh Blooms Delivered to Your Doorstep" with a call-to-action button
- **Featured Flowers** — Showcases three flower varieties in card layouts
- **Services Section** — Highlights Same Day Delivery, Fresh Flowers from Local Growers, and 24/7 Support
- **Hot Deal Banner** — Promotional banner with up to 25% off offer
- **Product Gallery** — Displays additional flower product images
- **Free Shipping Info** — Contact section with phone and email details
- **Newsletter Subscription** — Email signup with a $30 coupon incentive
- **Footer** — Brand name, tagline, and copyright notice

## Tech Stack

- **HTML5** — Semantic markup (`header`, `main`, `footer`)
- **CSS3** — Flexbox layout, custom pink/rose color scheme
- **Google Fonts** — Open Sans (400, 700, 800)
- **Font Awesome 6** — Icons (phone, envelope)
- **GitHub Pages** — Static hosting

## Getting Started

Clone the repo and open `index.html` in your browser:

```bash
git clone https://github.com/Ahammad204/Flower-shop.git
cd Flower-shop
start index.html
```
